## 关于我

自 2017 年加入 GitHub 以来，我持续构建以 **TypeScript** 和 **Python** 为核心的开发实践。近三个月活跃于 6 个公共仓库，主要贡献类型为代码提交（占比 85.7%），同时通过 issue 参与协作（占比 10.7%）。技术栈覆盖前端、脚本及移动端领域，包括 JavaScript、HTML、Java 与 Dart。目前在杭州 ManManMai 工作，专注于将想法转化为可运行的解决方案。
